🐾 ゆるゆる道案内
このファイルについて
Claude Code がプロジェクトを理解するための引継ぎ資料。
設計書 ver.1.4（2026年3月1日）の内容をまとめたもの。

アプリ概要
童謡「犬のおまわりさん」をモチーフにした、ゆるい徒歩道案内アプリ。
一生懸命だけどちょっとおせっかいな犬のおまわりさんが、ゆるく道案内してくれる。

技術スタック

フレームワーク: Streamlit
地図: streamlit-folium（Leaflet相当・クリックで座標取得可）
ルート取得: OpenRouteService（ORS）API
ランドマーク取得: Overpass API
天気取得: OpenWeatherMap API
AI セリフ生成: Groq API
APIキー管理: secrets.toml
ホスティング: Streamlit Community Cloud


キャラクター設定（犬のおまわりさん）

語尾に「〜ワン」をつける
お店・建物名はうろ覚えで、ぼんやりした表現のみ（正式名称は使わない）
ランドマーク情報は「〇〇っぽいとこの角」「〇〇みたいなとこの横」のように使う
交差点名はそのまま使ってよい（「〇〇交差点あたりで左に曲がるワン」）
距離感は「ちょっと」「けっこう」「すぐそこ」など感覚的に
一生懸命・おせっかい・寄り添い型のキャラクター


セリフ構成（2種類）
① 道案内セリフ
ORS の曲がり角情報 ＋ Overpass のランドマーク情報をもとに Groq が生成
② おせっかいセリフ
天気・気温・風・花粉・距離をもとに Groq が別途生成
※ 2つの Groq 呼び出しは Promise.all（Python では asyncio.gather）で並行実行

Overpass API フォールバック順

overpass-api.de（メイン）
overpass.kumi.systems（バックアップ）
maps.mail.ru（バックアップ）

各サーバー10秒タイムアウト。全滅時はランドマークなしで続行。

ランドマーク取得仕様
取得対象POI

コンビニ・スーパー（道沿い最優先）
ファストフード（マクドナルド・吉野家等）
ドラッグストア（マツキヨ・ウエルシア等）
携帯ショップ・100円ショップ（取れたらラッキー程度）
銀行
駅・バス停
学校・大きな病院・郵便局・警察署（大型施設）
公園・広場

距離フィルタ

小型店舗（コンビニ・ファストフード等）：曲がり角から 15m以内
大型施設（駅・病院・学校・公園・警察署・郵便局）：35m以内
大型施設を優先してソート（距離が遠くても先に出す）
曲がり角1箇所につき最大2件


Groq プロンプト方針
道案内セリフ

キャラクター設定：語尾〜ワン・距離感覚的・4〜5文
ランドマークは「〇〇っぽいとこの角・横・確認ポイント」として使う
交差点名はそのまま使ってよい
右折・左折は必ず正確な方向で案内（重要）
「重要」ステップは必須、「省略可」はスキップしてよい
「どん突き当たり」表現を使う

おせっかいセリフ

天気・気温・風・花粉リスク・距離感を渡す
2〜3文で出発前のアドバイス
押しつけがましくなく、やさしくおせっかいな感じで

Groq へ渡す曲がり角情報の例
1. 【重要】【どん突き当たり】【左折】左方向に進む（約80m） ／ 交差点名：和泉橋交差点 ／ 近く：コンビニ「ローソン〇〇店」
2. 【省略可】【右折】右方向に進む（約14m）
3. 【重要】【左折】左方向に進む（約200m） ／ 近く：駅「秋葉原駅」

画面遷移仕様
1画面目：スタート・ゴール入力

地図表示（ピン入力用）
テキスト検索ボックス（スタート・ゴール両方）
✕ボタンで個別クリア可
「🔀 入れ替え」ボタン（両方確定時のみ活性）
「検索」ボタン（両方確定時のみ活性）→ ORS完了後に2画面目へ遷移
ORSエラー時は1画面目にエラー表示して遷移しない
「🔄 最初からやり直す」ボタン

2画面目：ルート確認・道案内

スタート・ゴール地点名表示
ルート付き地図（fitBoundsで自動ズーム）
Overpass・天気取得中はスピナー ＋「ちょっと待つワン。今説明の順番を考えてるワン…」
取得完了 or タイムアウト後に「案内してもらうワン🐾」ボタンが出現
「案内してもらうワン🐾」押下 → Groq呼び出し（並行）→ セリフ表示
「🔄 最初からやり直す」→ 全消しして1画面目に戻る
「🐾 もう一回道案内してもらう」→ ルート・セリフのみリセット、スタート・ゴールは残す


API 呼び出しタイミング
タイミング処理「検索」ボタン押下ORS実行 → 完了後2画面目へ。同時にOverpass・天気を並行バックグラウンド取得開始2画面目バックグラウンドOverpass・天気が完了 or タイムアウト → 「案内してもらうワン🐾」ボタン出現「案内してもらうワン🐾」押下Groq（道案内＋おせっかい）を並行呼び出し

Streamlit 実装上の注意

ボタン押下のたびにスクリプト全体が再実行される
ORS・Overpass・天気の取得結果は session_state にキャッシュして再呼び出しを防ぐ
画面遷移は session_state の page フラグで管理（例：page=1 / page=2）
